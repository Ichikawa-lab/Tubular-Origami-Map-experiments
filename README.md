# Tubular-Origami-Map-experiments

Reproducible numerical experiments for the **Tubular Origami Map** implemented in Google Colab.

This repository reproduces the numerical experiments from our paper:  
**"Invariant Curves and the Variational Structure in Tubular Origami Dynamical Systems"** (arXiv:2603.02410)

https://arxiv.org/abs/2603.02410
